Jurassic Memory

Lors de ce projet, j'ai appris à coder en ruby sur une machine virtuelle sous Linux. Notre but était de créer une application pour répertorier des espèces de fleurs. Il est possible d'ajouter une fleur à la base de données, d'en modifier ou d'en supprimer. De plus, nous devions apprendre à utiliser l'IA générative d'images MidJourney pour illustrer les fleurs dont nous voulions parler sur l'application.
Taille de l'équipe : 2 personnes
Durée du projet : 1 semaine
Langage : Ruby
Applications : Virtual Box, Github
Rendu final : PetalPedia
