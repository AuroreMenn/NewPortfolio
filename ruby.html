Ruby on Rails

Lors de ce projet, j'ai appris à coder en ruby sur une machine virtuelle sous Linux. Notre but était de créer une application pour répertorier des espèces de fleurs. Il est possible d'ajouter une fleur à la base de données, d'en modifier ou d'en supprimer.
Taille de l'équipe : 2 personnes
Langage : Ruby
Applications : Virtual Box, Github
Rendu final : PetalPedia
