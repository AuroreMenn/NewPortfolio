Refonte du site du MAI de Nantes

Durée
2 semaines
Équipe
5 Personnes
Applications
Figma, Github, Visual Studio Code
Langage
HTML, CSS, JavaScript

Lors de ce projet, j'ai pu apprendre le maquettage de site web puis l'intégration en HTML CSS. Lors de la première semaine, j'ai réalisé une maquette car nous devions effectuer la refonte du site du Musée de l'Imprimerie de Nantes. J'ai donc appris des notions d'UX et d'UI, mais aussi d'accessibilité. Lors de la deuxième semaine, j'ai du intégrer le tout en HTML, CSS et JavaScript.

Rendu final : Refonte du site du Musée de l'Imprimerie de Nantes
